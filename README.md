# forge-and-fire-kt
📘 Warhammer Kill Team 資料與規則閱讀器<br>
⚠️ 本專案為非官方工具<br><br>
✨ 專案介紹<br>
這是一個針對「戰鎚：殺戮小隊」遊玩時使用的陣營資料與規則閱讀工具，以 手機／行動裝置瀏覽體驗優化為主。提供更直覺的部隊資料檢視與常用遊玩功能。<br>
本專案 非官方製作，所有資料均來自 Games Workshop 正式規則文檔，不做修改；若官方規則更新，會盡快同步更新專案內容。<br>
📋 功能特點<br>
🧠 手機友善的網頁介面設計<br>
🗂 針對不同陣營提供資料篩選與清單顯示<br>
🔍 武器規則可直接點選查看詳細說明<br>
❗ 可標記已使用的戰術（計謀）以便追蹤<br>
🔢 旁邊可記錄單位剩餘耐傷數<br>
📌 隱藏未上場單位，提升查閱效率<br>
<br><br><br>
殺戮小隊的各陣營都有販售實體的規則卡片，請多多支持臺灣戰鎚戰鎚店家並購買收藏，共同營造優質的遊玩環境。<br>
